# 深度阅读 Prompt 合集（整理版）

> 整合多个阅读模板，按「筛书 → 通读 → 精读 → 批判 → 应用 → 进阶」的流程组织，方便根据当前阶段快速选用。

---

## 一、筛书：快速判断一本书值不值得读

请快速分析《XXX》：

1. 这本书的核心价值是什么？
2. 为什么它出名？
3. 哪些内容最值得读？
4. 哪些内容可以跳过？
5. 适合什么人？不适合什么人？
6. 如果只花 1 小时阅读，应该重点看哪里？
7. 这本书属于「信息型 / 思维型 / 方法型 / 情绪型 / 营销型」中的哪一种？
8. 这本书是真有深度，还是"包装得很聪明"？

---

## 二、通用万能模板：真正读懂一本书

你现在是"深度阅读教练"。请帮我真正读懂《XXX》，不要只做摘要。

### 【第一部分：全局理解】

1. 这本书真正想解决什么问题？
2. 作者最核心的一句话是什么？
3. 整本书的逻辑结构是什么？

### 【第二部分：核心观点】

4. 提炼最重要的 5\~7 个观点
5. 每个观点的推理过程是什么？
6. 哪个观点最重要？为什么？
7. 观点之间是什么关系？哪个是根，哪个是枝叶？

### 【第三部分：底层逻辑】

8. 作者默认哪些前提成立？
9. 作者的世界观、假设、价值观是什么？
10. 作者对「人性 / 社会 / 商业」的根本看法是什么？
11. 他默认"什么是真的"？

### 【第四部分：推理与模型】

12. 作者是如何一步步推导出结论的？中间有没有逻辑跳跃？
13. 提炼书中的思维模型、公式、框架、决策方法。

### 【第五部分：现实应用】

14. 现实中如何使用？哪些场景特别有效？哪些场景会失败？
15. 生活怎么用 / 工作怎么用 / 商业怎么用 / 人际关系怎么用？

### 【第六部分：批判性思考】

16. 哪些人会反对这本书？哪些情况下这本书会失效？
17. 作者可能忽略了什么？
18. 它和哪些书思想接近？又与哪些书冲突？

### 【第七部分：精华浓缩】

19. 如果只记住 20%，那最关键的内容是什么？

### 【第八部分：互动训练】

20. 给我设计：3 个思考题、3 个现实案例、3 个训练任务。

**目标：** 让我不仅"知道"，而且真正"会用"。

---

## 三、章节拆解（精读单章）

请拆解《XXX》第 X 章。

输出：

1. 本章核心问题
2. 作者观点
3. 推理链
4. 关键概念
5. 最重要的一句话
6. 现实案例
7. 作者隐藏假设
8. 可能错误
9. 如何实际应用
10. 用普通人能听懂的话重新解释

---

## 四、按书籍类型分类阅读

### 4.1 哲学类

请帮我真正理解《XXX》的哲学思想，不要学术化。

1. 作者到底在思考什么问题？为什么这个问题重要？
2. 作者如何推导？
3. 与其他哲学家有什么区别？
4. 对现实人生有什么影响？
5. 普通人最容易误解哪里？
6. 如果真正理解，会如何改变一个人的生活？

### 4.2 心理学类

请分析《XXX》的心理学核心。

1. 人性的核心机制
2. 情绪运作方式
3. 行为模式
4. 潜意识影响
5. 常见认知偏差
6. 如何识别 / 如何改善
7. 现实案例
8. 容易被误用的地方

### 4.3 商业 / 赚钱类

请用"商业实战"角度拆解《XXX》。

1. 真正赚钱的逻辑
2. 商业模式
3. 用户心理
4. 竞争优势
5. 流量逻辑
6. 杠杆点
7. 风险
8. 哪些适合普通人 / 哪些只是幸存者案例
9. 如何低风险实践

### 4.4 小说 / 文学类

不要只讲剧情。请深度分析《XXX》：

1. 作者真正想表达什么？
2. 人物象征什么？故事隐喻什么？
3. 冲突背后的本质是什么？
4. 哪个人物最接近作者自己？
5. 作者对人性的看法是什么？
6. 为什么这个故事会打动人？
7. 如果放到现实世界意味着什么？

### 4.5 成长 / 思维类（拆书型）

请像"拆机器"一样拆解这本书。

输出：

- 输入条件
- 核心机制
- 运转逻辑
- 输出结果
- 成功条件 / 失败条件
- 副作用 / 被滥用后的风险

最后：用系统图方式描述整本书。

## 五、批判性思考

### 5.1 反洗脑：专门寻找漏洞

请不要默认作者正确。请专门寻找：

1. 作者最脆弱的逻辑 / 最大的逻辑漏洞
2. 被忽略的数据 / 缺乏证据的地方
3. 幸存者偏差
4. 时代局限 / 哪些观点现在已经过时
5. 情绪化论证 / 情绪大于事实的地方
6. 偷换概念 / 因果倒置
7. 哪些结论并不普适

最后判断：

- 这本书哪些部分可信？
- 哪些部分只能"参考"？

### 5.2 查理·芒格式多元思维分析

请用多元思维模型分析《XXX》。

从以下角度：

- 心理学
- 经济学
- 博弈论
- 概率论
- 进化论 / 生物学
- 系统论
- 人性

分析：

1. 作者哪里最深刻？哪里最容易误导人？
2. 哪些是普遍规律？哪些只是特定环境有效？
3. 哪些认知最值得长期保留？
4. 这本书最重要的"误判风险"是什么？

### 5.3 作者真实意图分析

不要只分析书的表层内容。请继续分析：

1. 作者真正想影响读者什么？
2. 作者最想改变读者哪个观念？
3. 作者有没有隐藏立场？
4. 作者是在描述世界，还是试图塑造世界？
5. 作者最大的偏见可能是什么？

## 六、从理解到应用

### 6.1 把书变成能力（实战系统）

不要继续讲理论。请把《XXX》转化成：

1. 可执行动作 / 日常训练 / 行动习惯
2. 每日训练
3. 检查清单
4. 决策流程 / 决策模板
5. 现实对话模板
6. 错误警报机制 / 风险提醒
7. 常见错误
8. 现实演练任务

**最终目标：** 让我在现实中真正使用这本书。

### 6.2 真正理解一本书的标志

如果一个人真的彻底理解了《XXX》，那么：

- 他会怎么看世界？
- 他会如何决策？
- 他会避免什么错误？
- 他和普通人最大的区别是什么？

再进一步：
什么人即使读完这本书，也依然不会改变？为什么？

---

## 七、高手进阶：读透一本书

### 7.1 五轮进阶法

**第 1 轮：建立全局地图**
先给我这本书的"思维地图"。

**第 2 轮：抓核心**
哪 20% 内容最重要？核心章节是第几章？这章为什么是全书核心？

**第 3 轮：攻击作者**
如果反对作者，最强反驳是什么？如果你是作者的敌人，你会如何反驳他？

**第 4 轮：现实化**
现实世界中，哪些公司 / 人物正在使用这套思想？

**第 5 轮：训练化**
帮我设计 7 天训练计划，让我真正掌握它。

### 7.2 最推荐的组合流程

**第一步：** 帮我建立《XXX》的思维地图。
**第二步：** 提炼核心模型与底层逻辑。
**第三步：** 攻击这本书最脆弱的部分。
**第四步：** 把它转化成现实行动。

---

## 八、压缩速查版

不要只总结这本书。而是：

分析作者的核心问题、底层假设、推理链、关键模型、现实应用、局限性、反对观点，并把它转化成我可以实际使用的能力。

**最终目标：** 让我拥有作者的思考能力，而不是只记住内容。
